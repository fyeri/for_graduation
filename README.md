# README

## 開発言語
- Ruby3.0.1
- Rails6.1.7.6

## 実行手順
```
$ git clone git@github.com:fyeri/for_graduation.git
$ cd goods_app
$ bundle install
$ rails db:create && rails db:migrate
$ rails s
```
